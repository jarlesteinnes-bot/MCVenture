üì± How do I start using MCVenture?

After downloading the app:

  • Grant location permissions when prompted
  • Choose your preferred language from 8 available options
  • Browse routes or tap "Track Ride" to start recording
  • Enable turn-by-turn navigation by selecting a route and tapping "Navigate"

üó∫Ô∏è How does turn-by-turn navigation work?

MCVenture provides voice-guided navigation:

  • Select any saved route or discovered route
  • Tap the "Navigate" button to start guidance
  • Voice announcements in your chosen language will guide you at 500m, 100m, and 30m before turns
  • The app automatically reroutes if you go off course
  • Toggle voice on/off using the speaker icon during navigation

üö® What is crash detection and how does it work?

MCVenture includes automatic crash detection:

  • Uses your device's motion sensors to detect sudden impacts
  • If a crash is detected, you'll see an alert asking if you're OK
  • If you don't respond, the app can alert emergency contacts
  • You can also manually activate SOS from the emergency button
  • Your location is shared only when SOS is activated

üîã Does MCVenture drain my battery?

MCVenture is optimized for battery life:

  • Uses battery-adaptive performance modes
  • Switches to low-power mode when battery is low
  • Pause tracking when stopped to save battery
  • Turn off voice guidance to extend battery further
  • On average, uses similar battery as other navigation apps
